Electronic Astronomy Resources

The following are based on a handout I received in a first-year grad-level astronomy/astrophysics course, with a few of my own additions.

Images

  • Sloan Digital Sky Survey (SDSS):
    A major multi-filter imaging and spectroscopic redshift survey.
    • Imaging query
    • Spectroscopy query
  • The Aladin Sky Atlas:
    An interactive software sky atlas allowing the user to visualize digitized astronomical images.
  • SkyView Virtual Observatory

Databases

  • Canadian Astronomical Data Center (CADC)
  • Centre de Données astronomiques de Strasbourg (CDS)
  • SIMBAD Astronomical Object Database (Harvard Site)
  • SIMBAD Astronomical Object Database (U-Strasbourg Site)
  • VizieR
    • Query page
  • NASA Extragalactic Database (NED)
  • Lyon-Meudon Extragalactic Database (LEDA)

Articles

  • Astrophysics Data System (ADS)
    • Abstract service
  • ArXiv - Astro-ph (preprint server)
  • Individual journals
    • The Astrophysics Journal (ApJ)
    • The Astronomical Journal (AJ)
    • Astronomy & Astrophysics (A&A)
    • Annual Reviews of Astronomy and Astrophysics (ARAA)
    • Monthly Notices of the Royal Astronomical Society (MNRAS)

Professional Associations

  • Canadian Astronomical Society (CASCA)
  • American Astronomical Society (AAS)
  • Royal Astronomical Society (RAS)
  • Royal Astronomical Society of Canada (RASC)
